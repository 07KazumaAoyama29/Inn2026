# 今日作るもの
今日作る作品は、**モンスターバトルゲーム**です。<br>

「こうげき」か「ひっさつ」を入力して、モンスターと戦います。<br>
プレイヤーとモンスターのどちらかのHPが0になるまで、バトルが続きます。<br>

```text
=== モンスターバトル ===

あなたのHP: 30
モンスターのHP: 40
行動を選んでください（こうげき・ひっさつ）: こうげき
あなたの攻撃！8ダメージ！
モンスターの攻撃！5ダメージ！

あなたのHP: 25
モンスターのHP: 32
行動を選んでください（こうげき・ひっさつ）: ひっさつ
必殺技！14ダメージ！
```

攻撃のダメージは毎回ランダムに変わります。<br>
必殺技は大きなダメージが出ることもありますが、外れて0ダメージになることもあります。<br>

# プログラム全体図
```text
① プレイヤーとモンスターのHPを決める

② プレイヤーとモンスターのHPを表示する
                  ↓
③ プレイヤーがコマンドを入力する
                  ↓
④ コマンドに合わせて攻撃する
                  ↓
⑤ モンスターのHPを減らす
                  ↓
⑥ モンスターのHPが残っていれば反撃する
                  ↓
⑦ どちらかのHPが0になるまでくり返す
                  ↓
⑧ 勝ち・負けを表示する
```

この作品は、Month1からMonth4までに学んだ内容を組み合わせて作ります。<br>

- 変数でHPを記録する
- `input`でコマンドを選ぶ
- `if / elif / else`で行動を変える
- `random`でダメージを決める
- `while True`, `break`でバトルをくり返す
- 条件式でバトルの終了を判断する

# 前回の復習
前回は、数字当てゲームを作りました。<br>

```python:demo.py
import random

num = random.randint(0,100)

while True:
  ans = input("数字を入力してください")
  ans = int(ans)

  if ans == num:
    print("正解！")
    break
  elif ans < num:
    print("もっと大きい数字です")
  else:
    print("もっと小さい数字です")
```

`while True`は、無限ループです。<br>
無限ループ中に`break`すると、ループが終わります。<br>
今回は、プレイヤーとモンスターのどちらかのHPがなくなるまで、バトルをくり返します。<br>

# リファレンス
今回のモンスターバトルゲームで使う、Pythonの基本的な書き方を確認しましょう。<br>

## 変数：データを覚えておく
変数は、数字や文字を保存する箱です。<br>
`変数名 = データ`の形で書きます。<br>

```python:python
# 整数を変数に入れる
player_hp = 30

# 文字列を変数に入れる
command = "こうげき"
```

変数の値を計算して、新しい値に変えることもできます。<br>

```python:python
# HPからダメージを引く
damage = 5
player_hp = player_hp - damage
```

## コメント：処理の説明を書く
行の先頭に`#`を書くと、プログラムの説明をコメントとして残せます。<br>
コメントは、プログラムを実行したときの処理には影響しません。可読性を高めるために使います。<br>

```python:python
# プレイヤーのHPを決める
player_hp = 30
```

## print()とstr()：文字や数字を表示する
`print()`を使うと、文字や変数の中身を画面に表示できます。<br>

```python:python
print("モンスターが現れた！")
print(player_hp)
```

文字列と整数を`+`でつなぐときは、`str()`で整数を文字列に変えます。<br>

```python:python
print("あなたのHP: " + str(player_hp))
```

## input()：文字を入力する
`input()`を使うと、キーボードで入力した文字を受け取り、変数に入れられます。<br>

```python:python
command = input("行動を選んでください（こうげき・ひっさつ）: ")
```

## 比較演算子：2つの値を比べる
条件を作るときは、次のような比較演算子を使います。<br>

| 書き方 | 意味 | 使用例 |
|---|---|---|
| `==` | 同じ | `command == "こうげき"` |
| `!=` | 同じではない | `command != "こうげき"` |
| `<` | 左の値が右の値より小さい | `player_hp < 0` |
| `>` | 左の値が右の値より大きい | `player_hp > 0` |
| `<=` | 左の値が右の値以下 | `monster_hp <= 0` |
| `>=` | 左の値が右の値以上 | `player_hp >= 30` |

`=`は変数にデータを入れる記号、`==`は2つの値が同じか比べる記号です。<br>

## if / elif / else：条件によって処理を変える
`if`を使うと、条件によって実行する処理を変えられます。<br>
上から順番に条件を調べ、最初に当てはまった処理だけを実行します。<br>

```python:python
if command == "こうげき":
  print("通常攻撃！")
elif command == "ひっさつ":
  print("必殺技！")
else:
  print("そのコマンドは使えません")
```

`if`、`elif`、`else`の行の最後には`:`を付けます。<br>
条件に当てはまったときに実行する処理は、行の先頭に半角スペースを入れて字下げします。<br>

## while Trueとbreak：処理をくり返す
`while True`の中に書いた処理は、そのままではずっとくり返されます。<br>
くり返しを終わらせたい場所に`break`を書きます。<br>

```python:python
monster_hp = 10

while True:
  monster_hp = monster_hp - 5

  if monster_hp <= 0:
    break
```

この例では、モンスターのHPが0以下になると`break`が実行され、`while`のくり返しが終わります。<br>
`break`は、`while`の中で使います。<br>

## random.randint()：ランダムな整数を作る
`random.randint(最小の数, 最大の数)`を使うと、指定した範囲からランダムな整数を1つ作れます。<br>
最初に`import random`を書いて、`random`モジュールを使えるようにします。<br>

```python:python
import random

# 5から10までの整数をランダムに作る
damage = random.randint(5, 10)
```

## 完成例
```python:main.py
# randomモジュールを使えるようにする
import random

# ゲームのタイトルを表示する
print("=== モンスターバトル ===")
print("あばれドラゴンが現れた！")

# プレイヤーとモンスターのHPを決める
player_hp = 30
monster_hp = 40

# 決着がつくまでバトルをくり返す
while True:
  # 今のHPを表示する
  print("")
  print("あなたのHP: " + str(player_hp))
  print("モンスターのHP: " + str(monster_hp))

  # プレイヤーの行動を入力する
  command = input("行動を選んでください（こうげき・ひっさつ）: ")

  # 入力されたコマンドで攻撃方法を変える
  if command == "こうげき":
    # 通常攻撃のダメージを決める
    damage = random.randint(5, 10)
    print("あなたの攻撃！" + str(damage) + "ダメージ！")

  elif command == "ひっさつ":
    # 必殺技のダメージを決める
    damage = random.randint(0, 15)
    print("必殺技！" + str(damage) + "ダメージ！")

  else:
    # 知らないコマンドでは攻撃できない
    damage = 0
    print("そのコマンドは使えません")

  # モンスターのHPを減らす
  monster_hp = monster_hp - damage

  # モンスターを倒したらHPを0にして、くり返しを終える
  if monster_hp <= 0:
    monster_hp = 0
    break

  # モンスターの攻撃ダメージを決める
  enemy_damage = random.randint(3, 8)
  player_hp = player_hp - enemy_damage
  print("モンスターの攻撃！" + str(enemy_damage) + "ダメージ！")

  # プレイヤーが倒れたらHPを0にして、くり返しを終える
  if player_hp <= 0:
    player_hp = 0
    break

# 最後のHPを表示する
print("")
print("あなたのHP: " + str(player_hp))
print("モンスターのHP: " + str(monster_hp))

# プレイヤーのHPを調べて勝敗を表示する
if player_hp > 0:
  print("あばれドラゴンを倒した！あなたの勝ち！")
else:
  print("力尽きてしまった……")
```
